<h1>機能概要</h1>
・ログイン、新規登録
・投稿（画像も可）、投稿削除
・プロフィール編集

<h1>制作の目的</h1>
今までインプットしてきた知識のアウトプットのために作りました。
身近に使っているアプリの方が完成のイメージしやすいと思い、twitterのクローンを作成することにしました。

<h1>使用している技術</h1>
PHP 7.4.9
MySQL 5.7.30
HTML5
CSS3
jQuery 3.3.1

mamp 6.0.1で開発しました。
